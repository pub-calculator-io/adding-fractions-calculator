Easily add and subtract proper, improper, and mixed fractions with our free Adding Fractions Calculator. Instantly solve math problems with up to 9 fractions!

[https://www.calculator.io/adding-fractions-calculator/](https://www.calculator.io/adding-fractions-calculator/)

== Usage ==

Add the Adding Fractions Calculator shortcode to your page, post or sidebar:

`[calcio_adding_fractions_calculator]`

Add the following code to your template where you would like the Adding Fractions Calculator to appear:

`<?php display_calcio_adding_fractions_calculator(); ?>`

== Installation ==

1. Upload the Adding Fractions Calculator /calcio_adding_fractions_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Adding Fractions Calculator plugin through the "Plugins" menu in WordPress.
